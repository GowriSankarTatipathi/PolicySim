# Bundle contents

This archive holds the artifacts generated in this session. It is NOT the
CASCADE/PolicySim source code repository (that was not present in this session).

- paper/
  - PolicySim_IEEE_Paper.tex   — full LaTeX source (IEEEtran, compiles to 16 pages)
  - PolicySim_IEEE_Paper.pdf   — compiled paper
  - IEEEtran.cls               — required IEEE class file
- PolicySim_Research_Proposal.md — earlier long-form research proposal
- research-guide.md              — general "how to write a strong research paper" guide
- github_templates/
  - .gitignore                 — production .gitignore for a Python research repo
  - gitignore.txt              — same file, .txt copy (in case .gitignore is hidden on your OS)
  - README_TEMPLATE.md         — README skeleton with all standard sections
- figures/
  - policysim_figures.html     — the PolicySim IEEE figures HTML you uploaded

To turn the templates into a real repo, copy github_templates/.gitignore and
README_TEMPLATE.md into your codebase, then adapt paths and fill placeholders.
